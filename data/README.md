# Data Directory

Use this directory for local or Azure data files used by SeaKR reproduction work.

Do not store datasets in Obsidian. Download datasets directly on the machine that will run the experiment.

- `raw/`: original downloaded archives and source files.
- `processed/`: derived files, extracted datasets, indexes, and preprocessing outputs.

Large data files are intentionally ignored by Git. Keep only lightweight notes and manifests under version control.
